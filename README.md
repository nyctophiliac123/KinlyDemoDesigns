# Kinly Demo Designs

This project contains landing page and splash screen designs created using **Google Stitch**.

## 🚀 Overview
A collection of high-fidelity prototypes and UI designs focusing on a refined, sanctuary-like user experience.

## 📂 Project Structure
* **stitch_kinly_landing_page**: Main landing page variations (refined, quiet, sanctuary).
* **stitch_kinly_splash_screen**: Mobile/Web entry screens.
* **kinly_gossamer**: Specialized design assets.

## 🛠️ Built With
* [Google Stitch](https://stitch.google.com) - Design & Prototyping
* [Google AI Studio](https://aistudio.google.com/) - Generative UI

## 📝 How to View
1. Clone this repository: `git clone https://github.com/nyctophiliac123/KinlyDemoDesigns.git`
2. Open the `code.html` files in any web browser to view the live designs.
